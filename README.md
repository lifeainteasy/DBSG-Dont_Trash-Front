# 프로젝트 참여 멤버
---
- 최종근 (프론트엔드, 백엔드)

# 만든 이유
- 대덕소프트웨어마이스터고등학교의 멋진 게시물을 홍보하고 싶어서

# 기능

---

- 로그인
- 회원가입
    - 동아리 부장, 동아리 회원, 동아리 관리자 선생님(지도 선생님),
- 메인페이지
    - 동아리 검색(선택)
    - 프로젝트 이름 검색(선택)
    - 사람이름 검색(선택)
    - 분야(웹, 앱, 게임,임베디드, 정보보안,데스크탑 앱, 기타 소프트웨어)
- 프로젝트 업로드
    - 프로젝트 규모
        - 게인
        - 동아리
        - 수업 시간 프로젝트 EX) 자바프로젝트, 임베디드 프로젝트
    - 프로젝트 이름
    - 프로젝트 개요
        - 프로젝트 배경과 개발 목표
        - 주요 기능
        - 서비스 시나리오 및 페르소나
    - 프로젝트 참여 인원
        - 이름
        - 대표사진
        - 깃허브주소
        - 페이스북주소
            - 추가 가능(인스타, 블로그)
        - 역할
    - 공개/비공개 설정가능
    - 개발일자
        - 달력
    - 마이페이지
        - 한줄소개
        - 한 프로젝트 목록
        - 깃허브 주소
        - 역할
    - 동아리 페이지
        - 동아리 소개
            - 동아리 이름
            - 동아리 목표
            - 동아리 프로젝트
            - 동아리 멤버
            - 동아리 개발 분야
        - 멘토링(보류)
            - 멘토링 신청
                - 멘토
                - 멘티
